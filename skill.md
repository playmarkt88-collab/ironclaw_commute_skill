# YandexTrainTracker

Description:
A custom utility skill that fetches real-time suburban train schedules between Bulatnikovo and Moskva-Tovarnaya using the Yandex Rasp API. 

Use Case:
Optimizes the commute routine by providing precise departure times directly within the agent interface, skipping heavy websites.

Execution:
Runs via the attached Python script. Requires YANDEX_API_KEY in the agent's environment secrets.
